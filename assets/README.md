# Assets

Place your application icons here:

- `icon.png` - Main application icon (256x256 or larger)
- `icon.ico` - Windows icon file (for the .exe)
- `tray-icon.png` - System tray icon (16x16 or 32x32)

## Creating Icons

You can use tools like:
- https://convertio.co/png-ico/ - Convert PNG to ICO
- https://www.aconvert.com/icon/png-to-ico/ - Online converter
